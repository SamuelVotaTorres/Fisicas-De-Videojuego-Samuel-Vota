# AUREN Physics Lab — Simulador guiado FUNCIONAL

Versión corregida para iniciar automáticamente y evitar bloqueo por pantalla inicial.

## Correcciones

- El simulador entra directo al módulo 1.
- Se corrigió la fórmula de colisiones que podía romper el JavaScript.
- El loop de simulación ya no depende de la pantalla inicial.
- Botones y teclado funcionan desde el inicio.
- Se agregó botón de ayuda rápida.

## Controles

- 1, 2, 3, 4, 5: cambiar módulo
- Q, E, F: acciones del módulo
- WASD: mover cubo verde cuando el módulo lo permite
- Espacio: impulso / salto del jugador
- R: reiniciar módulo
